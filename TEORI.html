PENJELASAN TEORI HTML



Bagian Head

  1. !DOCTYPE html:Deklarasi bahwa dokumen ini adalah dokumen HTML.
  2. html : Tag pembuka untuk elemen HTML.
  3. head : Tag pembuka untuk elemen head. Elemen ini berisi informasi tentang dokumen, seperti judul, charset, dan viewport.
  4. meta charset="UTF-8" : Menentukan charset yang digunakan dalam dokumen. UTF-8 adalah charset yang paling umum digunakan untuk website.
  5. meta name="viewport" content="width=device-width, initial-scale=1.0" :
    Menentukan viewport untuk website.Viewport adalah area yang terlihat di browser.Dengan pengaturan ini, website akan terlihat baik di berbagai perangkat,seperti desktop,laptop, dan smartphone.
  6. title Halaman Heading/title : Menentukan judul halaman. Judul ini akan ditampilkan di tab browser.
  7. /head : tag penutup untuk elemen head.

Bagian Body

  1. /li : body Tag pembuka untuk elemen body. Elemen ini berisi konten utama halaman, seperti teks, gambar, dan video.
  2. h1 : Halaman Heading Menampilkan heading 1 dengan teks "Halaman Heading". Heading 1 adalah heading terbesar dalam HTML.
  3. p : Ini adalah paragraf. Menampilkan paragraf dengan teks "Ini adalah paragraf."
  4. /body: Tag penutup untuk elemen body.
  5. /html : Tag penutup untuk elemen HTML.

Kesimpulan

Setiap elemen HTML harus memiliki tag penutup.

Tag penutup ditulis dengan cara yang sama dengan tag pembuka, tetapi dengan tambahan garis miring / di depan nama elemen.
contohnya : (h1)Heading(/h1)

Elemen void tidak memerlukan tag penutup.

Elemen void adalah elemen yang tidak memiliki konten, seperti (img) dan (br). Elemen void ditutup secara otomatis oleh browser.

Urutan penutupan elemen harus sesuai dengan urutan pembukaannya.

Elemen yang dibuka terakhir harus ditutup terlebih dahulu. Contohnya:
(div)
(h1)Heading(/h1)
(p)Paragraf(/p)
(/div)

Penutup yang tidak tepat dapat menyebabkan kesalahan pada halaman web.

Pastikan semua elemen HTML ditutup dengan benar. Kesalahan pada penutup HTML dapat menyebabkan masalah pada tampilan halaman web, seperti teks yang terpotong atau elemen yang tidak ditampilkan.
